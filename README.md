# My Alacritty Configuration
Backup/tracking for myself

## Setup
Make a backup of your current configuration (if it exists):

```bash
mv ~/.config/alacritty/ ~/.config/alacritty.bak/
```

Download and install the [Hurmit Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/Hermit.zip)

Clone this repo:

```bash
git clone https://github.com/jpisani0/alacritty-conf.git ~/.config/alacritty/
```

Consider forking this repo or removing the `.git` dir to make and commit your own changes
